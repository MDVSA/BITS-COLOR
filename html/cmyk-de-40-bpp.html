Bits y color

Profundidad en Bits y Modos de color

CMYK de 40 bpp




Supongamos que tenemos un archivo de 2 pix x 2 pix.

Se ocupan 8 bits (1 byte) para almacenar el número binario correspondiente a la cantidad de tinta cian,

8 bits para la tinta magenta, 8 bits para la amarilla,  8 bits para la negra y

un canal de 8 bits más (canal alfa) que permite codificar hasta 256 tipos de transparencia.
